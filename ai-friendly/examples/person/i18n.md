# Person Domain — i18n (pt-BR)

```typescript
// settings/locales/pt-BR.ts
export const ptBR = {
  person: {
    title: "Pessoa",
    fields: {
      id: "ID",
      name: "Nome",
      "name.placeholder": "Digite seu nome aqui",
      email: "E-mail",
      "email.placeholder": "Informe seu melhor e-mail",
      phone: "Telefone",
      "phone.placeholder": "Ex.: (11) 99999-9999",
      birthDate: "Data de Nascimento",
      active: "Ativo",
      street: "Rua",
      "street.placeholder": "Informe a rua do endereço",
      city: "Cidade",
      "city.placeholder": "Digite a cidade",
    },
    groups: {
      basic: "Informações Básicas",
      address: "Endereço",
    },
    actions: {
      custom: "Enviar",
    },
  },
};
```

## Key Patterns

1. **One entry per field** — `{domain}.fields.{fieldName}`
2. **Placeholders use dot notation** — `{domain}.fields.{fieldName}.placeholder`
3. **Groups need labels** — `{domain}.groups.{groupName}`
4. **Only custom actions need labels** — standard actions (add, view, edit, create, update, cancel, destroy) are in `common.*`
5. **Domain title** — `{domain}.title`

## Merge with core translations

```typescript
import { ptBR } from "@ybyra/core";       // common translations
import { ptBR as local } from "./locales/pt-BR";  // domain translations

const merged = { ...ptBR, ...local };
```
